# [MythicMobs Addon] MythicMobsPAPI

> Native: Java Enviroment 17 ~ 21 - MC Version 1.20.4 <br>

## Commands

> /mmpapi - Display help commands <br>
> /mmpapi info - List all usable placeholders <br>
> /mmpapi reload - Reload files <br>
